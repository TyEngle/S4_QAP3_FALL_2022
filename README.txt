QAP3 for Java class in semester 4 of Software Development 
The objective is to familiarize with the concepts of inheritance in OOP; Super(), extends, etc.
Two part program: 
  Part 1 involves creating people objects and displaying their attributes.
    Single parent class with multiple subclasses
  Part 2 involves creating shape objects and displaying their attributes.
    Single parent class with multiple subclasses, and a subclass off the subclass.
